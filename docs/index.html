Features

Everything you need,
nothing you don't.

No account. No server. No telemetry. Just a Chrome extension and your API key.

No bot in the call

Captures audio from your browser tab directly. No "AI Notetaker has joined" embarrassment.

Speaker diarization

Automatic speaker labels powered by OpenAI's gpt-4o-transcribe-diarize. Pin your own voice for stable names.

Works on any tab

Google Meet, YouTube, Loom, any browser audio. Not limited to a single platform.

Privacy first

Audio goes directly from your browser to OpenAI. No intermediary server. Audio deleted after transcription.

MCP for Claude

Native Model Context Protocol bridge. Ask Claude about your meetings without copy-pasting transcripts.

Origin

Paid $10/month for an API wrapper.

I was paying for a meeting transcription SaaS. One day I looked at the request in DevTools. It was sending my audio to the same OpenAI Whisper API I already had a key for.

So I built this in a weekend with Claude. Same quality, no subscription, no data leaving my machine except to OpenAI directly. Then I added an MCP bridge so Claude can read my meeting notes natively.

Total cost for a one-hour meeting: $0.36. That's the API call. The extension is free. Forever.
